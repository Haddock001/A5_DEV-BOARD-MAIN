Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element in DOM. In order to select an element by its ID we can use getElementbyID method. If we want to select multiple elements with class name we can use querySelectorAll method etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main differences are:
innerHTML: Includes html elements.
innerText: Includes text inside the HTML tag.
innerContent: Has no HTML tags and removes existing tags when setting a value.

Question-3: What is event delegation in the DOM?

So, basically while creating a website we may use many event in various elements individually. This may create problems while processing the website due to trying to process every event at once so the event delegations helps us to delegate actions or events by a specific time that we apply to keep the site running smoothly and flawlessly.

Question-4: What is event bubbling in the DOM?

In event bubbling a child element gets on top of its parent element and keeps going until eventually reaching the document.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element using document.createElement(), we can add elements by appendChild() and remove them by removeChild()
